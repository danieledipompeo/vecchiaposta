A) HTML Structure - top

This template is a fixed layout with two columns. The main content area has the class "main-content" (two thirds of the page area) and the sidebar area has the class "sidebar" (one third of the page area).

There are 2 primary fonts used in this template "Cardo" for titles and "Arial" for all other text. If you want to change them go to lines 128 and 134 in the "style.css" file. Note "Cardo" is a Google font, if you want to change to a different Google font don't forget to import it in the HTML head tags (see line 23 of any HTML file in this package)

B) CSS Files and Structure - top

C) JavaScript - top

This theme imports 9 Javascript files.

  1. jQuery
  2. jQuery UI
  3. jQuery UI Core
  4. jQuery UI Widget
  5. jQuery UI DatePicker
  6. prettyPhoto (photo gallery lightbox)
  7. Superfish (main menu drop down)
  8. Slides (homepage slideshow)
  9. All custom JS code is stored in scripts.js
  1. The only JS file you might need to edit is "scripts.js", for example if you want to edit the drop down menu, slideshow or prettyPhoto lightbox.
